How to Train Usiigaci
1. Install LOCI plugin on Fiji ImageJ
    Update Fiji: Help > Update
    Advanced Mode > Manage Update Sites > Checkbox LOCI > OK > Close > Apply Changes
    Restart Fiji
2. Manually generate ROI labels
    Load raw images into Fiji
    Analyze > Tools > ROI manager
    Use Freehand selection tool to draw outlines around cells
    Convert ROIs to label mask: Plugins > LOCI > ROI map
    Convert mask to 8-bit grayscale
    Save mask files into separate directory


How to Use Usiigaci
1. Download trained weights: https://www.dropbox.com/sh/3eldgvytfchm9gr/AAB6vzPaEf8buk81IRVNClUEa?dl=0
2. Edit inference.py
    (line 288) specify image directory
    (line 292) specify weights path
